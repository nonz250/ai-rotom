# ai-rotom

ポケモンチャンピオンズ対戦アドバイザー MCP サーバー。

## インストール・使い方

```bash
npx -y ai-rotom
```

MCP クライアント（Claude Desktop / Claude Code 等）の設定例、
提供される全ツールの一覧、ポケモンチャンピオンズ固有仕様の注意事項などは、
リポジトリのルート [`README.md`](../../README.md) を参照してください。

## 提供機能

AI クライアントから以下のような操作ができます:

- ポケモンの種族値・技・特性・持ち物情報の取得
- タイプ相性・実数値・素早さラインの計算
- 1 対 1 / 6 対 6 のダメージ計算
- パーティの弱点・攻撃カバレッジ分析
- 2 体対面 / 6v6 選出判断
- 対策候補 TOP 10 の抽出

## 要件

- Node.js >= 24
